1.インストールする 


1-1.解凍 

YesNoChart.zipを解凍し、テーマフォルダ直下にコピーします。 
例 ） 
wp-content/themes/サイト名/に解凍 
wp-content/themes/サイト名/YesNoChartフォルダが作成されます 


1-2.設定 

function.phpの最後に一文を追記します 
include "YesNoChart/plugin_YesNoChart.php"; 

php中で自動的にcss、jsの読み込みも行っています。 

以上です。 






2.使用する 

記事・固定ページにショートコードを記述する事で使用します。 


2-1個別作成：ページに直接記述する 

記事・固定ページ中にカスタムHTML、またはショートコードを追加します。 

プラグインにて追加されたショートコードを記述していきます。 

ページを表示してバグがなければ完成です。 


2-2使いまわす：固定ページに作成して読み込む 

固定ページを作成し、カスタムHTML、またはショートコードを追加します。 

プラグインにて追加されたショートコードを記述していきます。 

ページ保存時に非表示設定をします。 

他プラグイン等でnoindexをしていない場合、このプラグインで追加されるnoindex設定にチェックを入れてください。表示されていいない場合は、記事の設定→パネル→追加の「インデックス設定」をONにしてください。 

ページを表示してバグがない事を確認します。 

読み込みたい記事中で、 

[LoadYesNoChart Slug="ページスラッグ"] 

と記述します。 

エラーが発生しなければ完成です。 





2-3使いまわす：テキストファイル読み込み 

基本的にはこちらを推奨。

YesNoChartフォルダ以下の自由な場所にファイルを作成します。 

コードはJISではなくUTF8で保存してください。 

例 )  YesNoChart/sample/test.txt 

  

読み込みたい記事中で 

[LoadYesNoChartFile file="ファイル名"] 

※上記のファイルだとsample/test.txtと指定 

エラーが発生しなければ完成です。

4.ショートコード一覧 

固定記事の読み込み 

[LoadYesNoChart slug="固定ページのスラッグ"] 
  固定ページにショートコードでYesNoChartを書き、その固定ページを読み込む 


テキストファイルの読み込み 

[LoadYesNoChartFile file="ファイル名"] 
  フォルダ直下にショートコードを記述したテキストファイルを作成し 
  そのファイルを読み込む 


YesNoチャートの大枠 

[YesNo-Chart tag="タグ名" title="タイトル名" ex="補足説明"] 
    問題を記述 
[/yesNo-Chart] 
囲っている範囲をYesNoチャートにします 


問題 Question 一問目( 表示用クラスが付与されます ) 

[YesNo-Q1 id="href用固有ID" title="問題名" img="画像URL"] 
    選択肢を記述 
[/YesNo-Q1] 

 

問題 二問目以降 

[YesNo-Q id="href用固有ID" title="問題名" img="画像URL"] 
    選択肢を記述 
[/YesNo-Q] 

 

選択肢 ansower 

[YesNo-A href="ジャンプ先固有ID" text="選択肢名"] 
 

結果 Result 

[YesNo-R id="href用固有ID" title="診断結果名" img="画像URL" 
     btnhref="ボタンのa href" (省略時"#q1") 
     btntxt ="ボタン名" (省略時"もう一度診断する")] 
      
結果の文章をを記述します。
 
[/YesNo-R] 

 

引数のimgは簡易化する為に別パターンを指定できます 

img : ワードプレスのwp-content/以下のファイルを指定できます 
imgu: メディアライブラリのupload/以下のファイルを指定できます 
imgt: theme/現在使用中のテーマ/以下のファイルを指定できます 
i : YesNoChart/以下のファイルを指定できます。テキストファイルを読み込んだ場合は、テキストファイルのフォルダが基準なります。


ワードプレスに読み込まれているメディアライブラリのファイルを使用したい場合 
imguを使用、URLは記事にメディアライブラリから画像を埋め込み 
その画像をHTMLに変換して現れるURLです 


YesNoChartフォルダ内で完結したい場合 
iを使用します 


テーマフォルダ内の別フォルダから画像を読み込みたい場合 
imgtを使用します 

 




5.その他の仕様 

・インストールすると、記事編集時にNoIndex設定が表示されます 

表示されない場合は設定→パネル→追加→インデックス設定をON

　これは、YesNoChart用に固定ページを作った場合に

検索サイトに表示されないためのものです。






6.アンインストール方法 

・記事から記述を消します 

・function.phpから一文を消します 

include "YesNoChart/plugin_YesNoChart.php"; 

・フォルダを削除します 

  

以上です。
